# 🌌 NoCringe-Wallpapers Hub

A highly stylized, ultra-clean, and completely ad-free web hub for cyberpunk, minimalist, and cute setup wallpapers. Built with love !

🚀 **Live Demo:** [Launch Website](https://foura8.github.io/nocringe-wallpapers/)

---

## ⚡ Features

* 🚫 **100% Ad-Free & No Trackers:** No annoying popups, countdowns, or sketchy download links. Pure direct access.
* 🎨 **Cyberpunk Vibe:** Responsive, dark-themed grid layout optimized for modern desktop setups and mobile browsers.
* 📦 **Lightweight UI:** Built entirely on top of Tailwind CSS utilities for instant loading times.
* 🪙 **Native Crypto Modals:** Integrated, user-friendly modal overlays allowing seamless copy/paste actions for BTC and LTC tips.
* 📂 **Open-Source Structure:** Clean folder structures ensuring straightforward deployment on modern web hosts.

---

## 🛠️ Built With

* **HTML5:** Clean, semantic web scaffolding.
* **Tailwind CSS:** Modern utility-first styling for the neon neon-glow accents.
* **JavaScript:** Lightweight vanilla JS for reactive image modals and clipboard copy utilities.
* **GitHub Pages:** Free, globally distributed static site hosting.
